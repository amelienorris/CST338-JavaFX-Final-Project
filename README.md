# Chiikawa Task Pals

Chiikawa task pals is a JavaFX desktop application themed around the Chiikawa characters. Users can register and log personal tasks, milestones, and track focused work sessions.

Tech Stack:
- JavaFX
- SQLite
- OpenWeatherMap API
- Gradle
